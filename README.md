# House of Mit - Custom Design Studio

A MERN stack application that allows users to upload their own designs and visualize them on apparel models before ordering.

## Quick Start

1. Install dependencies:
```bash
cd backend && npm install
cd ../frontend && npm install
```

2. Start MongoDB (make sure it's running)

3. Start backend:
```bash
cd backend && npm run dev
```

4. Start frontend (new terminal):
```bash
cd frontend && npm start
```

5. Open http://localhost:3000

## Features

- Upload custom designs
- Regular Print & All Over Print (AOP) modes
- Drag-and-drop positioning
- Size adjustment and rotation
- Multiple products and colors
- Save designs to database
- Download final designs

For complete documentation, see the docs/ folder or contact support.
